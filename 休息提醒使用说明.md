# 休息提醒工具 - 使用说明

## 📋 简介

一个简单易用的定时休息提醒工具，帮助你在长时间工作时养成定时休息的好习惯，保护身体健康。

## ✨ 主要功能

- ✅ 自定义提醒时间间隔
- ✅ 多个快捷时间预设（30分钟、1小时、1.5小时、2小时）
- ✅ 弹窗强制置顶，不会错过提醒
- ✅ 友好的休息建议提示
- ✅ 贪睡功能（5分钟后再次提醒）
- ✅ 测试弹窗功能

## 🚀 快速开始

### 方式1：双击批处理文件（推荐）

1. 双击 `启动休息提醒.bat` 文件
2. 程序启动，设置提醒间隔
3. 点击"开始提醒"

### 方式2：运行Python脚本

```bash
python rest_reminder.py
```

### 方式3：使用桌面快捷方式

1. 双击 `create_shortcut.py` 运行
2. 桌面会出现"休息提醒"图标
3. 以后双击桌面图标即可启动

## 📖 使用教程

### 1. 设置提醒时间

- **手动输入**：在"提醒间隔"输入框中输入分钟数（如：45）
- **使用预设**：点击下方的预设按钮快速设置

### 2. 开始提醒

1. 设置好时间后，点击"开始提醒"按钮
2. 主窗口会最小化到任务栏
3. 继续你的工作，程序会在后台运行

### 3. 收到提醒

当到达设定时间后：
- 🔔 弹窗会自动弹出并置顶显示
- ☕ 显示友好的休息提示
- 💡 给出4条健康建议

**你可以：**
- 点击"我知道了"关闭提醒
- 点击"5分钟后"稍后再次提醒

### 4. 停止提醒

- 点击主窗口的"停止"按钮即可停止计时

### 5. 测试功能

- 点击"测试弹窗"可以随时预览提醒效果

## 🎯 健康建议

工具会随机显示以下建议：

- 🔸 站起来走动走动
- 🔸 远眺放松眼睛
- 🔸 喝杯水补充水分
- 🔸 做个简单的伸展运动

## 💡 使用技巧

### 推荐的休息时间

- **轻度工作**：每60分钟休息一次
- **高强度工作**：每30-45分钟休息一次
- **长时间工作**：每90分钟休息一次

### 休息时间做什么

1. **眼部放松**：远眺20秒，看窗外绿色植物
2. **身体活动**：站起来走动，做简单拉伸
3. **补充水分**：喝杯水或茶
4. **放松大脑**：闭眼深呼吸，听首轻松的音乐

### 20-20-20法则

- 每20分钟
- 看远处20英尺（约6米）
- 至少20秒

## 🛠️ 高级设置

### 自定义启动

编辑 `启动休息提醒.bat` 文件，可以修改启动参数。

### 开机自启（Windows）

1. 按 `Win + R`，输入 `shell:startup`
2. 将"休息提醒"快捷方式复制到打开的文件夹
3. 重启电脑后会自动启动

## ❓ 常见问题

### Q1: 双击批处理文件没反应？

**A:**
1. 检查是否已安装Python
2. 在命令行输入 `python --version` 测试
3. 确保rest_reminder.py文件在同一目录

### Q2: 提醒弹窗没出现？

**A:**
1. 检查是否被杀毒软件拦截
2. 确认程序没有被最小化到托盘
3. 点击"测试弹窗"功能测试

### Q3: 想修改默认时间？

**A:**
编辑 `rest_reminder.py` 文件，找到这一行：
```python
self.default_minutes = 60  # 修改这个数字
```

### Q4: 如何完全退出程序？

**A:**
直接关闭主窗口即可，计时器会自动停止。

### Q5: 可以同时运行多个提醒吗？

**A:**
不建议同时运行多个实例，可能会造成提醒混乱。

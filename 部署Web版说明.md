# 快速部署Web版休息提醒工具

## 方法1：GitHub Pages（推荐，免费）

### 步骤：

1. **进入仓库设置**
   - 访问：https://github.com/hujwCode/rest-reminder/settings/pages

2. **启用GitHub Pages**
   - Source: 选择 "Deploy from a branch"
   - Branch: 选择 "main" 和 "/ (root)"
   - 点击 "Save"

3. **等待部署**（1-2分钟）

4. **访问你的应用**
   - 访问：https://hujwCode.github.io/rest-reminder/rest_reminder_web.html
   - 或：https://hujwCode.github.io/rest-reminder/

## 方法2：Vercel（更快，免费）

1. 访问 https://vercel.com
2. 使用GitHub账号登录
3. 点击 "Import Project"
4. 选择 `rest-reminder` 仓库
5. 点击 "Deploy"
6. 几秒后获得访问链接

## 在手机上使用：

1. **Android (Chrome)**
   - 打开应用网址
   - 点击菜单 (⋮)
   - 选择"添加到主屏幕"

2. **iOS (Safari)**
   - 打开应用网址
   - 点击分享按钮
   - 选择"添加到主屏幕"

就这样！无需打包APK，直接在浏览器使用！
